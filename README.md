# ⌨️ Typer

A clean, interactive typing speed test application that measures your WPM (Words Per Minute) and accuracy in real-time.

## Features

- **Real-time WPM tracking** — see your speed update as you type
- **Accuracy measurement** — tracks correct vs incorrect keystrokes
- **Clean UI** — minimal, distraction-free interface
- **Instant feedback** — wrong characters highlighted immediately
